# Agent instructions

This repository is **Wizard Block of Truth** — a Swift/SwiftUI Wizard scorecard app with local multiplayer.

Before making changes, read the project skills under [`.cursor/skills/`](.cursor/skills/):

1. **`wizard-project`** — module layout, dependencies, build layout (start here)
2. **`wizard-domain`** — `GameCommand`, rules, constraints, scoring
3. **`wizard-multiplayer`** — host authority, guest permissions, wire protocol
4. **`wizard-ui-localization`** — SwiftUI views and `Localizable.xcstrings`
5. **`wizard-testing`** — test targets and `swift test`

Also see `localization.md` and `todos.md` for localization rules and product roadmap.
